What I am going to demonstrate on this page

This is my first web page. On this page I am going to demonstrate several different types of things. This is a Sample Paragraph with inline elements

I used the "abbr" tag so a screen reader for the blind doesn't try and pronounce the non-word as a word. I also used the "title=" attribute to define the abbreviation so it can appear as a screen tip or be read by the screen reader. Although you can style these elements in CSS as well, the HTML tags "strong" and "em" direct a screen reader to emphasize the word when it is being read. The CSS styling is strictly visual.
